# Security Policy

## Supported Versions

This is an early-stage project. Only the latest version on the default branch is supported.

## Reporting a Vulnerability

If you discover a security issue, please do not open a public issue with sensitive details.

- Preferably, report it via GitHub Security Advisories (if enabled for this repo).
- Otherwise, open a GitHub issue with minimal details and request a private channel.

Thank you for helping keep the project and its users safe.
